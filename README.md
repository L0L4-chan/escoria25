# Escoria Demo Game Refactor Plugin

## English

This project aims to **refactor the Escoria Demo Game plugin** and unify it into a single plugin.  
The main goals are:  
- Adapt it to allow the use of different UI options.  
- Simplify the overall project settings.  

This work is being developed as part of a **Bachelor’s Degree Final Project at UDC** and is currently **in progress**.

---

## Castellano

Este proyecto pretende **refactorizar el plugin Escoria Demo Game** y unificarlo como un único plugin.  
Los principales objetivos son:  
- Adaptarlo para permitir la utilización de diversas opciones de UI.  
- Simplificar los ajustes generales del proyecto.  

Este trabajo se realiza como **Trabajo de Fin de Grado en la UDC** y se encuentra actualmente **en desarrollo**.
